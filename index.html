CSS Simp

box sizing:

content-box= contents is the same as the width and height. border-box= boder + padding + content. The window will shrink the content to fit in height and width box Display properties: block: takes up 100% of the width and starts on a new line. example p and div tags. Inline: doesn't allow you to set a width or height for elemets and start on the same line. ex span and img tags.

Postion:

Static(default): follows noramal flow of the page. Relative: follows normal flow of the page. Lets you position using top, left, bottom, right. Absolute: does not follow the normal flow of the page. Lets you position with top, left, bottom, right. Postions is based off closest 'position: relative' parent. Uncommon ones, Fixed: sticks element to a permanent loctaion on the page. Lets you use top, right, bottom, left. Sticky: sitck elements to a permaneent location on the page once it hits a threshold.

Media queries:

used to make a web page responsive. Make sure the highest media is at the top in styles sheet
    Some numbers to remember
  • mobile phone: 576px
  • Tablet: 768px
  • Desktops: 992px
  • Large Desktops: 1200px+

Pseudo selectors in CSS:

after places things after a section/text. Before places things before a section/text. Pseudo element is accessed by ::. Pseudo class is accessed by : To add a transition, you only need to add it to the parent element.

Animations:

create it use @keyframes and then name the animation. Need a starting point 0% and an end point 100%. Staring point needs to either transform at the X or Y axis and add by how many pixels. In the parent element you need to add animation: name of animation, how long it last and either infinite(but jumps back to start) or infinite alternate-reverse(continous) Without infinite, then just give it a number of how many times you want it to happen and the animation with stop once it reaches that number

BEM(Block-Element-Modifier):

naming convention used for CSS classes to make them easier to read and understand, easy to maintain and change.
  • B [Block] is the block component. example; button, nav, main, footer(name of the massive section)
  • E [Element] is dependent on the block component. example; footer, link
  • M [Modifier] is used to change the style of the block. example; footer, link visited: not = white, visited = purple

CSS Notes

Start with a container
Display flex: puts elements side by side. But it also unlocks flex-direction, align-items and justify-content.

Flex-Direction:

by default its in a row. Set flex-dirction to column if you want coulumns

Align-items:

by defult it is stretched to the top and bottom on the container so it uses the vertical axis(cross axis). Main one is align-center, it centers elements vertically. flex-end, alings elements to the bottom of container.

justify-content:

by default flex-start aligns elements horizontally(main axis). center, puts elements in the center of the container. flex-end, puts elements to end of container on the horizontal axis. Space-around makes sure the space between the sides of the container and in between the elements are exactly the same. sapce-between, moves elements all the way to the sides and make sure the space between the elements are the same.

Using all three:

Flex-direction: column = justify-content becomes vertical axis and align-itmes becomes horizontal axis
Flex-flow: using flex-dirction and flex-wrap. You can also use calc to push things to differnet lines. For example 6 boxes, width: calc(100% / 3) will only have 3 boxes on a line and push the others to the next line.

What does parents and children mean?:

  • Parents refers to any element with other elemets inside it.
  • Child refers to an element inside of another element
Main axis is the horizontal axis. It controls how elements are laid out horizontally. The Cross axis is the vertical axis. How the elements are laid out vertically. When using flex box, it is concerned with the parent and then controls how the children are laid out.
